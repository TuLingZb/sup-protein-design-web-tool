# 载入结构后自动生成
运行前须知
  • contig 中带链号（A1-30）表示保留输入残基，纯数字表示新生成残基，/0 表示链断裂。单链模式不含 /0。
  • 下载的“输入 PDB”已按片段设为独立链 A/B/C…，并将每段蛋白残基重编号为 1..N，与 contig 索引一致——请把它作为 input 传给 RFD3。
  • 先用 prevalidate_inputs=True 验证；确认安装的化学组分库能识别 IUM，否则改用仅蛋白条件设计再回填铀酰坐标。
  • RFD3 只负责骨架/linker 生成，不校验铀配位化学；输出需按 U–O 距离/角度、位点供体、片段连续性筛选后再做 MD 最小化。
